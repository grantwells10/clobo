This repository contains the implementation prototype for Clobo, a mobile borrowing platform built with React Native and Expo. To reproduce our results, please see the below:

### Prerequisites: 
* Node.js (v18+ recommended)
* npm or yarn
* Expo CLI tools (installed automatically when using npx expo)

You will need either of the following (we used Expo Go app)
* Expo Go app on a mobile device
* iOS Simulator (Xcode)
* an Android Emulator
  
### Installation

    git clone https://github.com/grantwells10/clobo
    cd clobo
    npm install
(Or run yarn install.)

### Running the App

    npm expo start

Then choose one of the following:
* Press i to open the iOS simulator
* Press a to open the Android emulator
* Scan the terminal QR code using the Expo Go app

If you encounter missing module errors:

    npx expo install

No backend setup or API keys are required.
